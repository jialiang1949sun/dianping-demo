# 使用说明（最终版模板）

本项目为“简化版大众点评”应用，提供商户浏览、搜索筛选、评价、收藏、商户后台等功能。

## 访问地址

- 网站地址：<填写你的公网 URL>
- API 健康检查：`/api/health`

## 账号与角色

- 普通用户：可浏览、收藏、写评价、查看个人资料/我的评价。
- 商家用户（merchant_admin）：可进入商户后台，创建/编辑店铺、上传封面与相册、回复评价。
- 管理员（admin）：拥有商家能力，并可对所有商户执行管理类操作（当前实现以接口权限为准）。

## 主要功能

### 1) 首页

- Hero 轮播 + 热门分类 + 为您推荐（按分类分组展示）
- 附近好店：需要允许浏览器定位

### 2) 搜索与筛选

- 关键词搜索：商户名称/地址
- 分类筛选：动态分类
- 筛选：最低评分、距离排序、距离范围（需要定位）

### 3) 商户详情

- 基本信息：分类、地址、电话、营业时间、简介
- 地图：基于经纬度展示
- 店铺相册：展示商家上传的相册图片
- 评价：查看评价、登录后可写评价（1-5 星，评论 ≤200 字）
- 收藏：登录后可收藏/取消收藏

### 4) 个人中心

- 个人资料：昵称/手机号、头像上传
- 我的收藏：查看收藏列表、取消收藏
- 我的评价：分页查看历史评价

### 5) 商户后台（商家账号）

- 店铺信息管理：创建/编辑店铺、定位经纬度、封面图
- 封面上传：上传图片并保存
- 店铺相册：批量上传/删除、相册图一键设为封面
- 评价回复：在店铺详情页对评价进行回复/编辑

## 常见问题

- 无法定位：检查浏览器定位权限是否允许；HTTPS 环境下定位成功率更高。
- 距离筛选无结果：商户需填写经纬度；或距离范围过小。
